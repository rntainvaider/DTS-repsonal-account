## Запуск проекта
1. Создать виртуальное окружение:
```
>> python -m venv venv
```
2. Создать файл в корне проекта `.env` с перемеными
   - `USERNAME`
   - `PASSWORD`
   - `LOCALHOST`
   - `PORT`
   - `DBNAME`
3. Установить зависимости backend:
```
>> pip install -r requirements.txt
```
4. Данные команды запускают сразу frontend и backend:
```
>> cd frontend
>> npm install
>> npm start
```
